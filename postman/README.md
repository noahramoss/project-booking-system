# Colección de Postman

Exporta tu colección de Postman y guárdala en esta carpeta como:

```
booking-system.postman_collection.json
```

## Cómo exportar desde Postman

1. Abre Postman
2. Haz clic derecho en la colección
3. Selecciona **Export**
4. Elige el formato **Collection v2.1**
5. Guarda el archivo en esta carpeta
